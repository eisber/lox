CONNECTION
The preferred cable for connecting this sensor is a STP (Shielded Twisted Pair) cable, though UTP (Unshielded Twisted Pair) can be used. The use of a STP cable is especially important in environments with high EMI. The shielding must be grounded to be effective. Before connecting, check that the supply voltage is correct according to the operating voltage specified in this datasheet.
