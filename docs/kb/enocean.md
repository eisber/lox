## TECHNICAL DATA
- 1x EnOcean Transceiver
- Allows connection with up to 128 EnOcean Devices
- Configure with Loxone Config
- Connection via Loxone Link
- Dimensions: 35.5 x 88 x 57 mm (2 TE)
- Power supply 24VDC
- Power consumption is approximately 30mA
- Protection: IP20
- Ambient temperature: 0 to 50°C

*[EnOcean Extension Layout]*

					**Please note:** The repeater function of the EnOcean protocol is not supported.

## INSTALLATION
- Mount extension on a DIN rail
- Connect 24V DC power supply, but do not switch on until the whole installation is completed

(We recommend looping the power through the Miniserver)
- Connect the Loxone Link
- Note: If it’s the last extension, terminate the Loxone Link with a 120Ω resistor

(Included with the Miniserver)
- Turn on the power supply
- After a few seconds, the left LED will flash green if the extension is already configured, or orange if not
