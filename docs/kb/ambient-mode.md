# Ambient Mode

Source: https://www.loxone.com/enen/kb/ambient-mode/

---

**Ambient Mode** in the Loxone App is a customizable, always-on dashboard designed for tablets, desktops, and web interfaces in landscape view. It displays key information such as weather, time, and date, along with quick access to selected components. You can also navigate through the app directly from Ambient Mode. This feature is ideal for wall-mounted or stationary devices and can be personalized.

## Ambient Mode Explanation

To open Ambient Mode, please open the App menu and select “Ambient Mode”. Here is the Ambient Mode legend:
- Exit Ambient mode
Activate Screensaver
Settings
- Setting always launch app in Ambient Mode
- Default Screen (Favourites, Central, Room, Category,…)
- Choose background image
- Screensaver
- System Status Messages
- Searching tool
- Open Default screen of Ambient mode
- Weather, time and current date (Weather data only available with active [Weather service](https://shop.loxone.com/enen/search-result?query=Weather+service))
- All components of selected Default screen
- Quick navigation through Loxone App
![Ambient mode 3](https://www.loxone.com/enen/wp-content/uploads/sites/3/2025/08/Ambient-mode-3.jpg)

## Requirements:
- Ensure your Loxone App and Loxone Config are updated to version 14.x or later, where Ambient Mode first became available.
- Device must be used in landscape orientation only (tablets, desktop, web interface). Portrait mode is not supported in this view.
- Minimum resolution of the device must be 1024×700.
