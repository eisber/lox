# LiveView Monitoring Tool

Source: https://www.loxone.com/enen/kb/liveview-monitoring-tool/

---

With our monitoring tool, you can keep all important values on the screen even whilst changing pages to help error-finding in your configuration logic.
The monitor is activated in LiveView by marking an element on the config page and clicking the green pin on the upper left of the screen.

*[LiveView Monitor Pin]*

Use the plus / minus icons to show or hide the desired values.

*[LiveView Monitor Plus]*

The selected values are displayed in the grey box. The monitor can be closed with the grey minus sign in the upper left corner.

The values can be selected not only from a page but from the whole project, provided that specific values have been selected. In our example “AQ” or “Q” from Temperature Sensor.
