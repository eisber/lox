# App

Source: https://www.loxone.com/enen/kb/app/

---

With the function block App, an app installed on the smartphone or tablet can be called directly from within the Loxone App.

## Table of Contents
- [Properties](#Property)
- [Basic Programming](#Description)
- [Documents](#documents)

---

## Properties

| Summary | Description | Default Value |
| --- | --- | --- |
| URL | URL for the app call e.g.: spotify:// | - |
| Icon | Icon for mobile app By default the icon of the app is used | - |
| Symbol color | Color of the symbol in the user interface | - |
