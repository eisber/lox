# Min Max

Source: https://www.loxone.com/enen/kb/min-max/

---

#### Find the largest and smallest value from 4 analogue values.

## INFORMATION

The values of the four inputs connected to AI1 to AI4 are evaluated and compared and then at the output AQmin the lowest value is output, and at AQmax the highest value is output.

If an input of the Min Max function block is not connected then the value of 0 will be included in the calculation. To get around this problem you can simply duplicate one of the inputs and connect it to the spare input on the block.
