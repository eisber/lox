## WEB INTERFACE

The WebSocket client or Web Interface allows bidirectional communication between the web browser and Mini Server. This displays states and state changes in real-time. The client can only be used in conjunction with a WebSocket-enabled browser. For best results use Google Chrome.

The WebSocket is at the address of your mini server accessible:

http: // address-miniserver /

## APPS FOR IPHONE, IPAD AND ANDROID

For owners of iOS and Android devices, we offer native apps to connect to the Mini Server.

#### DISPLAY LAYOUT ANDROID

For Android devices defined by the device manufacturer, whether the HD version of the app is used based on the Configuration Objects:

SCREENLAYOUT_SIZE_SMALL

normal app

SCREENLAYOUT_SIZE_NORMAL

normal app

SCREENLAYOUT_SIZE_LARGE

HD App

SCREENLAYOUT_SIZE_XLARGE

HD App
