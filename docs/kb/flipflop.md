# Flipflop

Source: https://www.loxone.com/enen/kb/flipflop/

---

## INFORMATION

After a rising edge at input set S, the output Q is activated. The state remains stored until the input reset R is triggered. The input R is dominant – so that if both S and R are on then the output Q is disabled.

The block can be set to be retentive, this is done by clicking on the battery symbol   to set the red NOT dot on the block (input is by default inactive).
