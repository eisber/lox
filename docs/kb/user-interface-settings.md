# User Interface Settings

Source: https://www.loxone.com/enen/kb/user-interface-settings/

---

## USER INTERFACE

The User Interface Settings option allows the user through the app to make basic changes to how and where various items are displayed in the UI.

To access the User Interface Settings when in any Room, Category or Function within the Loxone app simply click on the “three dots” icon in the top right of the screen as shown below.

![Expert Mode 1 jpg 574x1024 iphone6 silver portrait 152x300](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/10/Expert_Mode_1_jpg-574x1024_iphone6_silver_portrait.png)

![no subject    james mccubbin loxone com   Loxone Mail 1 572x1024 iphone6 silver portrait 152x300](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/10/no_subject__-_james_mccubbin_loxone_com_-_Loxone_Mail-1-572x1024_iphone6_silver_portrait.png)

The option will then be presented to open the User Interface settings.

![UI Settings 2 576x1024 iphone6 silver portrait 152x300](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/10/UI-Settings-2-576x1024_iphone6_silver_portrait.png)

For Rooms or Categories it is possible to then change the:
- Description
- Icon
- Rating
- If the room is a favourite

![IMG 1829 576x1024 iphone6 silver portrait 150x300](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/10/IMG-1829-576x1024_iphone6_silver_portrait.png)

For functions you may edit the:
- Description
- Rating
- If the function is a favourite (and will, therefore, appear on the “Favourites” page in the app)
- Assigned Room
- Assigned Category

![UI Settings 3 576x1024 iphone6 silver portrait 150x300](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/10/UI-Settings-3-576x1024_iphone6_silver_portrait.png)

## CONFIG SETTINGS

In order to allow access to the User Interface Settings option within the app the logged in user must be part of a user group that has been allowed access rights to the User interface settings.

To do this in config simply select the User Group that wish to give access to and check the box for “User Interface Settings” in the properties panel as shown below. For more info on Users and User Groups, [click here!](https://www.loxone.com/enen/kb/users-groups/)
