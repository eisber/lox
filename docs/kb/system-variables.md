# System Variables

Source: https://www.loxone.com/enen/kb/system-variables/

---

System Variables are the next step in weather management and weather reactive control. Currently you have been stuck using the Weather Service/ Weather Station with little option otherwise. With version 10, you are now able to access this data. You can use the variables as outputs in your configuration or you can specify your own values.

## Use in Config

To use the System Variables in Loxone Config, simply select the variable you want to receive data from and drag it into config. You can select the source for the data in the properties of the variable.

*[]*

#### Using a third party Weather Source

In order to specify your own data for these variables, simply select the chosen variable and click ‘Define value by Logic’ in the title bar.
