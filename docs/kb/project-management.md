# Loxone Config Start Page and Project Management

Source: https://www.loxone.com/enen/kb/project-management/

---

The start page of Loxone Config provides tools to manage and access Loxone projects on your computer. It also offers a variety of additional features to streamline your workflow.
When opening Loxone Config, the **start page** allows you to open and manage the Loxone projects on your computer.

Additional features and tools are also available.

## Project Management

[
![Project Management 15.5](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/03/Project-Management-15.5.png)
](https://www.loxone.com/enen/kb/project-management/project-management-15-5/)
- Displays all saved Loxone projects along with their details.
- Create a new project.
- Launches the Project Planning tool.
- Opens File Explorer to load Loxone project files (*.loxone).
- Opens a Miniserver Keychain with all saved credentials.
- Opens a Miniserver search and lists all Miniservers found on the network.
- Opens a window to manually connect to a Miniserver.
- Manage and download new firmware, e.g. for the Audioserver.
- Allows you to [reset the password](https://www.loxone.com/enen/kb/restore-factory-defaults-format_sd-card/#password_recovery) to log in to the Miniserver.
- Opens a dialog to [format an SD card](https://www.loxone.com/enen/kb/restore-factory-defaults-format_sd-card/) for Loxone devices.
- Display Message Center, where you receive information that is directly relevant to you and your partner status.
- Display a change log of the latest versions of Loxone Config.
- All saved Loxone projects or found Miniserver are listed here.
- Use this field to search for a specific project or Miniserver.
- Allows you to log in with your Loxone account, e.g. to access Exosphere Workspace.
- Menu for additional options and file path of the selected project.

## New Project

To create a new program, simply select the type of Miniserver:

![Project Assistant new project 15.5](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/03/Project-Assistant-new-project-15.5.png)

The new project will then open, allowing you to begin programming.

## Project Backups

Project Backups can be created and saved at any time.

You can later select a backup and restore the project to this state.

First select the project in the periphery tree, and click on **Project Backups**:

![Project Backups 15.5.](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/03/Project-Backups-15.5..png)

The Project Backups window enables you to open, delete, restore, and create backups:

![Project Backups 15.5. 2](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/03/Project-Backups-15.5.-2.png)

To create a new backup:
- Click **Create new backup**.
- Add a note describing the current project status.
- Click **Create**.

![Project Backups 15.5. 3](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/03/Project-Backups-15.5.-3.png)

When a backup is opened, it appears as a grayed-out project page in Loxone Config.

Click on **Restore** to restore the project to this backup state:

![Project Backups 15.5. 4](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/03/Project-Backups-15.5.-4.png)

The backup is then imported into the open project, and you can continue working with it.
