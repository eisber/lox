# Wiping Relay

Source: https://www.loxone.com/enen/kb/wiping-relay/

---

## INFORMATION

After a rising edge at input Tr, the output Q is active for the duration of TH. The output pulse is restarted with each rising edge at the input Tr.

The block can be set to be retentive, this is done by clicking on the battery symbol   to set the red NOT dot on the block (input is by default inactive).
