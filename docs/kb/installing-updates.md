# Installing Miniserver Updates

Source: https://www.loxone.com/enen/kb/installing-updates/

---

The Miniserver update can be performed locally or remotely. For this, a user of the full access group must connect to the Miniserver in Loxone Config.
The Miniserver update can be performed in 3 different ways:

[Update via Loxone Config](#update_config)

[Update via App](#update_app)

[Update via formatting the SD card](#update_formatSDcard)

## Update via Loxone Config

1. [Connect to the Miniserver](https://www.loxone.com/enen/kb/initial-setup-miniserver/#connect_to_ms) and select “Miniserver” in the periphery tree. Click on “Miniserver Firmware Update” in the menu bar:

*[]*

2. You can now choose which data should be saved from your Miniserver. Click on “Install Update” to start the update:

*[]*

After the update, the Miniserver reboots and the update is complete.

If the update fails, preform the following steps:

**Restart update:** There may have been a temporary network problem or an error during the transfer. It is very likely that if you try to update again, everything will be successful.
- **Update via [formatting the SD card](https://www.loxone.com/enen/kb/restore-factory-defaults-format_sd-card/#format_sd):** if option 1 failed

## Update via App

Certain versions of the software are also published by Loxone via the app with a few days delay, if it contains important updates to the security or function of the overall system.

Users with the right “Miniserver Update via App” will then receive a notification via the system status, and after confirmation they can start the Miniserver update via the app.

*[]*

## Update via formatting the SD card

By [formatting the SD card,](https://www.loxone.com/enen/kb/restore-factory-defaults-format_sd-card/#format_sd) the Miniserver can be updated.
