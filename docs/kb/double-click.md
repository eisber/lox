# Double Click

Source: https://www.loxone.com/enen/kb/double-click/

---

## INFORMATION

With two successive pulses (a double click) at the input Tr then a pulse occurs at the output Q.

Parameter M allows you to set how long the interval between the two pulses needs to be (so you can change the speed of the double click). The duration of the pulse at the output is set by the parameter T.

A pulse at the input R causes the output to be reset to 0.
