# Parameter Templates

Source: https://www.loxone.com/enen/kb/parameter-templates/

---

FUNCTION
Using parameter templates, the parameters of a program block can be transferred easily to other blocks of the same type. Parameters can be easily modified for several blocks in a project by changing only one parameter template.

BASIC PROGRAMMING

The parameter templates for the relevant block type can be managed in the block properties.

*[Parameter Templates]*

All the parameter templates created so far are listed for the selected block type. You can create new templates with Save As and existing templates can be deleted and reloaded and renamed here.

For example, you can create a parameter template for all blinds with the same travel time.  You can make changes centrally to a module and these are copied for all objects with this parameter template.
