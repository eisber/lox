# Backup SD Card

Source: https://www.loxone.com/enen/kb/backup-sd-card/

---

The SD Backup feature creates copies of every file that is currently on the SD card at the time. These can then be copied to your computer and then stored on any device you choose.

## CREATE SD BACKUP

First ensure that the Miniserver is selected in the periphery. Then under the Miniserver tab on the top ribbon simply click “Backup SD Card”. If you are using a client/gateway system then both SD cards will be backed up.

![EN KB Config SD backup 3](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/04/EN-KB-Config-SD-backup-3.png)

In the SD Card backup window you will have the option to select which elements you want to create a backup of. All of these are checked by default.

![EN KB  Config SD BACKUP2](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/04/EN_KB__Config_SD_BACKUP2.png)

During data backup you will have an overview of what is currently happening. In our case, the 2nd of 9 configuration files are currently backed up.

![EN KB  Config SD BACKUP3](https://www.loxone.com/enen/wp-content/uploads/sites/3/2017/04/EN_KB__Config_SD_BACKUP3.png)

When the backup is complete a message will appear stating “Backup Completed” and you can then close the window.
